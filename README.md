# 🛍️ Customer Shopping Behavior Analysis

An end-to-end data analytics project exploring customer shopping behavior — covering Python-based EDA, SQL-driven data querying, and an interactive Power BI dashboard to uncover purchasing patterns, customer segments, and business insights.

## 📌 Project Overview

This project analyzes customer shopping behavior data to identify trends in purchasing patterns, customer demographics, and spending habits. The goal is to help businesses make data-driven decisions around marketing, inventory, and customer retention strategies.

The workflow follows a complete data analytics pipeline:
1. **Business Problem Definition**
2. **Data Collection & Cleaning**
3. **Exploratory Data Analysis (Python)**
4. **Data Querying & Analysis (MySQL)**
5. **Dashboard & Visualization (Power BI)**

## 🎯 Business Problem

Detailed in [`00_Business_Problem/Business Problem Document.pdf`](./00_Business_Problem/Business%20Problem%20Document.pdf), this project addresses the need to understand customer shopping patterns to support better business decision-making around sales, marketing, and customer experience.

## 🗂️ Dataset

The dataset (`customer_shopping_behavior.csv`) contains customer transaction and demographic data, including attributes such as customer demographics, purchase amounts, product categories, payment methods, and purchase frequency.

---

## 🐍 Python Analysis
`02_Python/Customer_Shopping_Behavior.ipynb` covers:
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Statistical summaries & visualizations
- Key pattern and trend identification
**Tools used:** Python, Pandas

## 🗄️ SQL Analysis
`03_MySQL/Customer_Behavior_sql_queries.sql` includes queries for:
- Customer segmentation
- Sales & revenue analysis
- Product/category-level insights
- Aggregations and business KPIs
**Tool used:** MySQL

## 🚀 How to Use This Repository
1. Clone the repo:
```bash
   git clone https://github.com/<your-username>/<your-repo-name>.git
```
2. Explore the business problem in `00_Business_Problem/`.
3. Load `01_Dataset/customer_shopping_behavior.csv` for your own analysis.
4. Run `02_Python/Customer_Shopping_Behavior.ipynb` in Jupyter Notebook.
5. Execute queries from `03_MySQL/Customer_Behavior_sql_queries.sql` in your MySQL environment.
6. Open `04_PowerBI/Customer_Behavior_Dashboard.pbix` in Power BI Desktop to explore the dashboard.
